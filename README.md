**EduCodes** is a web-based MVP (Minimum Viable Product) designed as a prototype for an educational startup business plan. The platform aims to teach programming concepts through interactive visual simulations rather than just text.

Although this version serves as a frontend prototype without a persistent database, it features fully functional logic simulators that demonstrate algorithmic concepts in real-time.
